# Power Pages Helper

## 简体中文

Power Pages Helper 是一个浏览器扩展，旨在增强 Microsoft Power Pages 的用户界面和体验。

### 功能

*   **显示版本号:** 在 Power Pages 的页眉中间区域显示当前扩展的版本号，方便您快速了解正在使用的版本。
*   **优化表格布局:** 自动调整表格中特定列的宽度，并移除父容器的最大宽度限制，使表格内容更易于阅读和理解。
*   **改善滚动体验:** 调整页面的滚动容器样式，提供更流畅、更自然的滚动效果。
*   **禁用列表虚拟化:** 通过模拟滚动操作，强制加载列表中的所有项目，从而禁用虚拟化，确保您能够一次性查看和操作所有数据。

---

## English

Power Pages Helper is a browser extension designed to enhance the user interface and experience of Microsoft Power Pages.

### Features

*   **Display Version Number:** Displays the current extension version in the center region of the Power Pages header, allowing you to quickly know which version you are using.
*   **Optimize Table Layout:** Automatically adjusts the width of specific columns in tables and removes the max-width restriction on parent containers, making table content easier to read and understand.
*   **Improve Scrolling Experience:** Adjusts the styles of the page's scroll container to provide a smoother and more natural scrolling effect.
*   **Disable List Virtualization:** Disables virtualization by simulating scrolling to force load all items in a list, ensuring that you can view and interact with all data at once.
